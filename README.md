# i2i Academy - Containerization

This repository contains the solution for the i2i Academy Containerization assignment.

## Theoretical Knowledge

1. Containerization packages an application with all its dependencies into a single isolated unit, ensuring it runs consistently across any computing environment.
2. This technology is essential to prevent environment-related bugs and to simplify the overall deployment process.
3. While Docker is the industry standard due to its ease of use, alternatives like Podman offer daemonless security, and LXC provides lightweight system-level virtualization.
4. In real-world scenarios, containerization is highly beneficial for microservices and cloud-native applications because it allows rapid scaling and independent updates.
5. It is also crucial in CI/CD pipelines to guarantee identical environments for development, testing, and production.
